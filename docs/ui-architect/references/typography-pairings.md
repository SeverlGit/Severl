## Pairing Principles

1. **Contrast in structure, harmony in tone.** Pair a geometric sans with a humanist serif, not two geometric sans faces.
2. **Shared x-height.** Fonts that pair well have similar x-heights so they align visually at the same size.
3. **Weight distribution.** If the display font is heavy (700+), keep the body font at 400. If the display font is light (300), body can go to 500.
4. **Mono for data.** Always use a monospace font for numbers, metrics, code, timestamps. It's not just aesthetic — it's functional (columns align).
5. **Max 3 fonts.** Display + body + mono. Never more. Two is often better (display + mono, using display at different weights for body).
